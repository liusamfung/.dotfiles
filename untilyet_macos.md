## Browser

### Extensions

[Time Tracker - Web Habit Builder](https://addons.mozilla.org/en-US/firefox/addon/besttimetracker/) Website timer limitator (AMAZING)
[uBlock Origin](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/) Add blocker
[UnTrap for YouTube](https://addons.mozilla.org/en-US/firefox/addon/untrap-for-youtube/) Video Quality setter, highly customizable YouTube page, Short blocker, etc etc etc.
[YouTube Auto Pause/Play](https://addons.mozilla.org/en-US/firefox/addon/youtube-auto-pause-play/) Don't do multitasking!!!!
[1password](https://addons.mozilla.org/en-US/firefox/addon/1password-x-password-manager/) Password manager
[Dark Reader](https://addons.mozilla.org/es-ES/firefox/addon/darkreader/) Change light theme to customizable dark theme on any webpage
[PhotoShow](https://addons.mozilla.org/es-ES/firefox/addon/photoshow/) View FullHD image on hover mouse, shotcut to download and copy HD imagen link. View format of image. Like QuickLook(MacOS).
